OBS Misa Translator v0.5.116 / README draft

配信者向けリアルタイム字幕・コメント翻訳ツール

OBS Misa Translator は、マイク音声の文字起こし・翻訳字幕、コメント翻訳、OBS Text Source / Browser Source 出力をまとめて扱う配信支援ツールです。

主な機能

リアルタイム字幕
faster-whisper による音声認識と、翻訳字幕のOBS出力。
字幕HTML演出
字幕00〜04を個別のBrowser Sourceとして表示。フェード、ポップ、タイプライター風など。
コメント翻訳
YouTube / Twitch / わんコメ連携を想定したコメント取得・翻訳・表示。
CSSデザイン
コメント・字幕のフォント、色、背景、枠、影、行間、表示位置を調整。
GPU対応
CUDA環境が利用可能な場合、GPUで高速な文字起こしが可能。
保存最小化
コメント本文の永続保存を避け、リアルタイム表示・翻訳に寄せた設計。

使用ガイド

基本的な流れ

  1. マイクを選択
  2. 音声認識モデルを選択
  3. CPU / GPU を選択
  4. 翻訳先言語を選択
  5. 「字幕翻訳開始」を押す

翻訳結果は OBS Text Source または Browser Source へ出力されます。

おすすめ初期設定

  • GPU環境あり: large-v3-turbo + GPU
  • 軽量環境: small / medium + CPU
  • 翻訳先: English
  • 字幕HTML演出: Fade または Pop

字幕TEXTソース設定

  • 自動消去秒数
  • 1行文字数
  • 最大行数
  • 途中送信

OBS Text Source向けの字幕整形設定です。

字幕ブラウザソース設定

  • フォント / サイズ
  • 文字色 / 背景色 / 枠色
  • 影 / 縁取り / 角丸
  • CSSテンプレ / アニメーション

Browser Source字幕の見た目を調整できます。

ブラックリスト / セーフリスト

  • ブラックリスト: 除外したい語句
  • セーフリスト: 除外を上書きして通す語句

セーフリストはブラックリストより優先されます。

クイックスタート

  1. Python環境と必要ライブラリを準備します。
  2. 仮想環境 venv_obsmt を有効化します。
  3. OBS_Misa_Translator_v0_5_116.py を起動します。
  4. マイク、音声認識モデル、CPU/GPU、翻訳先言語を設定します。
  5. OBS側へ txt または HTML の出力先を登録します。
cd /d D:\WorkFolder\OBSMisaTranslator
venv_obsmt\Scripts\activate
python OBS_Misa_Translator_v0_5_116.py

OBS設定

Text Source

OBSのテキストソースで、各 obs_subtitle_00.txt〜obs_subtitle_04.txt を読み込みます。

  • 字幕00: 認識結果
  • 字幕01: 翻訳結果
  • 字幕02〜04: 追加翻訳字幕

Browser Source

字幕HTMLを使う場合は、OBS Browser Sourceに専用HTMLを登録します。

subtitle_overlay_00.html
subtitle_overlay_01.html
subtitle_overlay_02.html
subtitle_overlay_03.html
subtitle_overlay_04.html

コメント機能

コメント取得・翻訳・OBS表示を行います。コメント本文や投稿者名を外部に永続保存しない方針へ寄せています。

わんコメ連携案

  • 初期実装候補: 翻訳をBOTコメントとして新規投稿
  • 将来候補: コメントID一致による既存コメント更新
  • 注意: 更新時はWordPartyが発火しない場合があります。

データ保存方針

コメント本文・投稿者名などのプラットフォーム由来データは、原則としてOBSMT側で永続保存しない方針です。表示・翻訳・読み上げなど配信中の一時処理を中心に扱います。
保存しない方針
  • コメント本文ログ
  • 投稿者ごとの長期履歴
  • 外部DB/クラウド保存
一時利用
  • OBS表示用txt/js/html
  • 配信中メモリ
  • 本文を含まない集計データ
