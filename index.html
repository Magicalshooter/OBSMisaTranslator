  • フォント / サイズ
  • 文字色 / 背景色 / 枠色
  • 影 / 縁取り / 角丸
  • CSSテンプレ / アニメーション

音声キャリブレーション

  • マイクの環境音と話し声を測定します。
  • 音声認識の開始しきい値の参考値を確認できます。
  • マイク入力が小さすぎる、または反応しすぎる場合の調整に役立ちます。

コメントブラウザソース設定

  • コメント表示のフォント、色、背景、枠、影を調整できます。
  • コメント表示件数や左右配置、アニメーションを変更できます。
  • OBS Browser Source向けの見た目を設定する画面です。

コメントTEXTソース設定

  • OBS Text Source向けのコメント表示形式を調整できます。
  • 表示件数、1行文字数、最大行数を設定できます。
  • 左右配置はOBS側のテキストソース設定で調整してください。

コメント整形・フィルタ詳細

  • URL、長文、空白コメントなどの扱いを設定できます。
  • 読み上げ最大文字数やOBS表示最大文字数を調整できます。
  • スパムや不要なコメントを減らしたい場合に使用します。

カスタム絵文字辞書

  • YouTube絵文字などのショートコードを登録できます。
  • 表示する文字列や画像URLを指定できます。
  • コメント表示を配信向けに整えたい場合に便利です。

読み上げ辞書

  • コメント読み上げ時の読み方を登録できます。
  • 略語、ネットスラング、固有名詞の読みを調整できます。
  • VOICEVOX読み上げを自然にしたい場合に使用します。

インストール / Installation

1. Downloader を起動

  1. Booth またはitch.ioから OBS Misa Translator Downloader.exe をダウンロードします。
  2. Downloader を起動します。
  3. 「Download / Install」を押します。

Downloader は最新版を自動取得し、installer を起動します。

2. Installer を実行

  1. ライセンスを確認します。
  2. インストール先を選択します。
  3. Install を押します。

初回起動時は Windows Defender の確認が出る場合があります。

3. OBS Misa Translator を起動

  1. マイクを選択
  2. Whisperモデルを選択
  3. 翻訳先言語を選択
  4. 「字幕翻訳開始」を押します

更新はヘッダーの「更新」ボタンから行えます。

音声認識の頭の良さ（Whisper モデル管理）

OBS Misa Translator は faster-whisper モデルを利用します。
Whisperは翻訳システムの一種で、モデルとは頭の良さのようなものです。

  • small: 軽量
  • medium: バランス型
  • large-v3-turbo: 高品質

モデル管理画面から不要モデルを削除できます。

Whisper モデルは数GBになる場合があります。

初回利用時は Whisper モデルのダウンロードが行われます。 モデルサイズによっては数GBになる場合があります。 モデル管理から先にダウンロードしておくとスムーズです。

コメント機能

コメント取得・翻訳・OBS表示を行います。コメント本文や投稿者名を外部に永続保存しない方針へ寄せています。

わんコメ連携案

  • 初期実装候補: 翻訳をBOTコメントとして新規投稿
  • 将来候補: コメントID一致による既存コメント更新
  • 注意: 更新時はWordPartyが発火しない場合があります。

アップデート

OBS Misa Translator は自動更新確認に対応しています。

  • 起動時に最新版確認
  • ヘッダー「更新」ボタン通知
  • installer自動ダウンロード
  • SHA256検証対応

データ保存方針

コメント本文・投稿者名などのプラットフォーム由来データは、原則としてOBSMT側で永続保存しない方針です。表示・翻訳・読み上げなど配信中の一時処理を中心に扱います。
保存しない方針
  • コメント本文ログ
  • 投稿者ごとの長期履歴
  • 外部DB/クラウド保存
一時利用
  • OBS表示用txt/js/html
  • 配信中メモリ
  • 本文を含まない集計データ

Speech Recognition Quality (Whisper Model Manager)

OBS Misa Translator uses faster-whisper models.
Whisper is the speech recognition system, and the model is like its level of intelligence.

  • small: lightweight
  • medium: balanced
  • large-v3-turbo: high quality

You can delete unused models from the Model Manager.

Whisper models may take several gigabytes of storage.

On first use, OBS Misa Translator downloads the selected Whisper model. Some models are several gigabytes. Downloading them in advance from Model Manager can make setup smoother.

Comment Features

OBS Misa Translator can fetch, translate, and display comments. The design avoids persistent storage of platform-derived data such as comment text and author names.

OneComme Integration Plan

  • Initial implementation candidate: post translated text as a new BOT comment.
  • Future candidate: update existing comments by matching comment IDs.
  • Note: WordParty may not trigger when comments are updated instead of newly posted.

Updates

OBS Misa Translator supports automatic update checks.

  • Check for the latest version on startup
  • Update button notification in the header
  • Automatic installer download
  • SHA256 verification support

Data Storage Policy

As a general policy, OBSMT does not persistently store platform-derived data such as comment text or author names. It focuses on temporary real-time processing for display, translation, and text-to-speech during streams.
Not Stored Persistently
  • Comment text logs
  • Long-term per-author history
  • External database or cloud storage
Temporary Use
  • OBS display txt/js/html files
  • In-memory data during streams
  • Aggregate data that does not include comment text
